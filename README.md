# HR Analytics Dashboard

## 📊 Project Overview
This project analyzes employee performance, KPI scores, attendance, and attrition risk using Power BI.

## 🛠 Tools Used
- SQL
- Excel
- Power BI

## 📈 Key Insights
- Identified high attrition risk employees
- Analyzed department-wise performance
- Compared KPI vs Performance Rating

## 📁 Files Included
- Dataset (Excel)
- Power BI Dashboard (.pbix)
- Dashboard screenshots
